# 💻 Portfolio Project

Персональное веб-приложение для демонстрации проектов и навыков. Проект базируется на **React 18** и использует современный инструментарий для сборки и стилизации.

## 🛠 Технологический стек

### Core
* **React (v18.3.1):** Библиотека для создания пользовательских интерфейсов.
* **React Router DOM (v6.23.1):** Декларативная маршрутизация для одностраничных приложений.
* **Web Vitals:** Мониторинг ключевых показателей производительности (метрики Google).

### UI & Styling
* **Tailwind CSS:** Утилитный фреймворк для быстрой адаптивной верстки.
* **PostCSS:** Трансформация стилей (включая автопрефиксы и оптимизацию).
* **Sass (SCSS):** Использование препроцессора для расширенных возможностей CSS.

### Build & Tooling
* **Webpack 5:** Мощная система сборки модулей и ресурсов.
* **Babel:** Транспиляция JavaScript кода для поддержки различных версий браузеров.
* **Fast Refresh:** Мгновенное обновление компонентов в браузере без потери их состояния при разработке.
* **Workbox:** Поддержка PWA (Progressive Web App) функционала и кэширования.

### Quality & Testing
* **Jest & Testing Library:** Полный цикл тестирования компонентов и логики.
* **ESLint:** Проверка качества кода с конфигурацией `eslint-config-react-app`.

## 🚀 Команды для запуска

В корневом каталоге проекта вы можете запускать следующие команды:

### `npm start`
Запускает приложение в режиме разработки по адресу `http://localhost:3000`.

### `npm run build`
Создает оптимизированную сборку проекта для продакшена в папке `build`.

### `npm test`
Запускает интерактивный прогон тестов.

### `npm run deploy`
Выполняет автоматическую сборку и деплой приложения на **GitHub Pages**.
